# Python Tutor Agent — Simplified Curriculum & Assessment

## Overview
- **Teaching model:** Lessons → Topic quizzes (MCQ) → Section tests (MCQ) → Level final (MCQ).
- **For now:** No code execution. All assessments are multiple-choice.  
- **Future (planned):** Live IDE + compiler for code-writing tasks with automated scoring.

---

## Levels, Sections, and Topics

### Beginner (B)
**Sections:** 4 | **Topics:** 10

- **B1 · Foundations**
  1. Python mindset, scripts vs REPL
  2. Variables & built-in types (int, float, str, bool)
  3. Operators & expressions (arith/comp/logical)

- **B2 · Control Flow**
  4. Strings & common methods (slicing, f-strings)
  5. Conditionals (if/elif/else)

- **B3 · Data & Functions**
  7. Lists & tuples (indexing, slicing, mutability)

- **B4 · Reliability & Libraries**
  9. Basic errors & debugging mindset
  10. Standard library tour (math, random, datetime)

## Teaching & Assessment Flow

### Per Topic
- **Lesson:** Short theory (why it matters → key ideas → example → pitfalls → recap).
- **Quiz:** 4 MCQs (concepts + small snippet reasoning).

### Per Section
- **Test:** 15 MCQs covering all section topics.

### Per Level
- **Final Test:** 30 MCQs across the full level.

---

## Scoring & Progression

- **Topic quiz pass:** ≥ 70%  
- **Section test pass:** ≥ 75%  
- **Level final pass:** ≥ 80%

**Overall Level Score (weights):**
- Topic quizzes: 20%  
- Section tests: 40%  
- Level final: 40%

**Advancement Rule:** Unlock next level if  
- Level final ≥ 80% **and** section average ≥ 75%.

**Retakes:**
- Topic quizzes: unlimited (best score kept).  
- Section tests: 1 retake (best kept).  
- Level final: 1 retake after review set.

---

## Review & Remediation (for now)
- If Topic < 70% → show mini-remedial explainer + allow re-quiz.  
- If Section < 75% → assign 2 remedial cards per weak topic → retake test.  
- If Level final < 80% → schedule review week before retake.
